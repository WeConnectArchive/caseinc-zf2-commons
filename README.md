# CASE zf2-commons

## Installation

```json
{
    "repositories": [
        {
            "url": "git@github.com:caseinc/zf2-commons.git",
            "type": "git"
        }
    ],
    "require": {
        "caseinc/zf2-common": "dev-master"
    }
}
```

Install dependencies

	php composer.phar install
